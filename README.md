\## Project Description

This project is an audit of VLC Media Player, an open-source multimedia player developed by the VideoLAN project. It explores its origin, license, Linux usage, ecosystem, and comparison with proprietary software. It also includes shell scripting tasks.



\## Scripts Included

1\. System Identity Report

2\. Package Inspector

3\. Disk and Permission Auditor

4\. Log File Analyzer

5\. Manifesto Generator



\## How to Run



Give permission:

chmod +x \*.sh



Run scripts:

./script1\_system\_identity.sh

./script2\_package\_inspector.sh

./script3\_disk\_auditor.sh

./script4\_log\_analyzer.sh /var/log/syslog error

./script5\_manifesto\_generator.sh



\## Dependencies

\- Linux OS

\- Bash

\- VLC Media Player



Install VLC:

sudo apt install vlc

This project demonstrates understanding of open-source software and practical Linux skills using shell scripting.
